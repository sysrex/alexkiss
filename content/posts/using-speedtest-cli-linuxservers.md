### Install speedtest-cli

{{<highlight bash>}}

sudo apt-get install python-pip
sudo pip install speedtest-cli

{{</highlight>}}



### And now we test

{{<highlight bash>}}

speedtest-cli

{{</highlight >}}


If you wish to share you results just run with:


{{<highlight bash>}}

speedtest-cli --share

{{</highlight >}}
